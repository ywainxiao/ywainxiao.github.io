总结一下 Python 代码语句的特点：

1.利用缩进来区分语句块，就是同样缩进的相邻语句组成一组语句块；
2.用 「:」 来引出需要执行的语句；
3.每个语句后面不需要分号或者其他结尾的符号。

函数简介

函数可以说是每一种语言中都会有的重要的概念，使用函数的目的是把一些经常要用到的功能写成一个整体，以后直接调用函数就可以实现相应的功能。函数实现的机制是输入-输出，输入一个或多个参数，在函数体内执行相应的操作后输出一个参数（我们称作返回值）。
